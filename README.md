# microsat-util
A micro satellite data processing toolkit

# Features

## Georeferencing

[WIP] Georeference an image
```
$ georef <input_file> -o <output_directory>
```

Extract KeyPoints
```
$ georef-keypoints <input_file> -o <output_file>
```

[WIP] Match Images
```
$ georef-match <file1> <file2> -o <output_file>
```
